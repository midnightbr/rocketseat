Imagens

Desafio proposto pelo curso introdutorio de html da rocketseat. Nesse desafio usaremos apenas html, já que até o momento foi o foco de assuntos.
Mais pra frente entrarão as outras tecnologias para linguagem web, como CSS e JavaScript.

Até o momento estou gostando bastante do curso, por ser bastante de dadico, o que ajuda muito alguém que está iniciando na programação.
Explicando cada tag e suas funcionalidades, o que acaba sendo bastante util na hora que começarmos a programa páginas mais complexas e desenvolver projetos.

Como dito no curso, um bom programador tem que saber o que está acontecendo com seu código e o porque de usar aquelas funcionalidades.
As vezes tem uma funcionalidade que poderá da o mesmo resultado na parte do cliente, ou seja, visualmente pode parecer a mesma coisa, no entanto para o navegador com certeza a diferença.
Tags diferentes representam coisas diferentes para o navegador e para o servidor. Algumas coisas podem ajudar bastante tanto na parte de front-end quanto no back-end.
Isso acaba sendo uma boa prática na programação estruturar seu código de forma correta e coerente.
